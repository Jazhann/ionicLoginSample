# Ionic Login Sample

Ionic login sample with unit and e2e tests

## Installation

Clone the repository, you need node.js installed and execute

```bash
npm install
```

## Usage

Build desktop version
```bash
npm run build
```

Build mobile version
```bash
npm run build_mobile
```
